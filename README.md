# Sabores Salvadoreños 🇸🇻🍛
> **Tour Gastronómico de El Salvador**

Una plataforma web interactiva y responsiva dedicada a promover el turismo gastronómico en El Salvador. Permite a los usuarios explorar rutas culinarias emblemáticas como la Ruta de la Pupusa, la Ruta del Café y la Ruta del Mar, además de contar con un sistema de contacto integrado.

---

## 🚀 Características

* **Navegación Responsiva:** Menú adaptable (mobile-first) con efecto funcional mediante CSS (Checkbox Hack) y transiciones suaves.
* **Diseño Visual Atractivo:** Sección *Hero* con imagen de alta calidad a pantalla completa (`100vh`) y superposición de capas (*overlay*) para mejorar la lectura.
* **Sección de Tours:** Rejilla dinámica (*CSS Grid*) que organiza tarjetas de servicios con efectos interactivos (`hover`).
* **Formulario Funcional:** Formulario de contacto estilizado con **Bootstrap 5** y conectado a la API de **StaticForms** para envíos en tiempo real sin recargar la página.
* **Experiencia de Usuario (UX):** Uso de notificaciones emergentes (*Toasts* de Bootstrap) para confirmar el envío exitoso del formulario.

---

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructura semántica del sitio.
* **CSS3 Personalizado:** Estilos globales, layouts con Grid/Flexbox y animaciones.
* **Bootstrap 5.3.8:** Implementación de componentes para el formulario, espaciados y el sistema de notificaciones (*Toast*).
* **JavaScript (Vanilla JS):** Manejo asíncrono de eventos con `Fetch API` para procesar el formulario de contacto.
* **FontAwesome:** Iconografía para redes sociales.
* **StaticForms API:** Servicio backend para la recepción y procesamiento de correos.
